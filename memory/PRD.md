# Alberto Filgueiras PhD - Professional Website

## Problem Statement
Professional website for Alberto Filgueiras, sports psychologist offering consulting, mental coaching, cognitive training, a brain performance program, published books, and an online course.

## Pages (Current - 7 pages)
1. **Home** — All-in-one page: Hero (with champion photo), popup 10% off, stats, logo carousel, features ("Inteligência Emocional"), credentials, education, experience carousel + 13 institutions, achievements, YouTube videos, research areas, CTA, bio ("Quem Sou Eu"), books
2. **Serviços** — 3 service types with WhatsApp CTAs, benefits, FAQ, books
3. **Programa Cérebro em Alta Performance** — 16-session program, science-backed
4. **Livros** — 4 books with full descriptions and Amazon links
5. **Curso Psicologia e Processos Básicos** — 12 YouTube lectures from UERJ (2020-2022)
6. **Contato** — Contact form, WhatsApp, Instagram, LinkedIn
7. **Login** — Dark themed (mocked)

## Architecture
- Frontend: React + Tailwind CSS + Shadcn UI
- Backend: FastAPI + MongoDB (not yet implemented)
